COMMON ISSUES & SOLUTIONS
==========================

Issue: "Application failed to start"
Solution:
  - Check logs: railway logs
  - Verify DATABASE_URL set
  - Verify EMAIL_HOST_USER and PASSWORD set
  - Run migration: railway run python manage.py migrate

Issue: "Static files not loading (500 error on CSS/images)"
Solution:
  - Check STATIC_URL and STATIC_ROOT in settings
  - Collect static files: python manage.py collectstatic
  - Verify WhiteNoise middleware in MIDDLEWARE list

Issue: "Emails not sending"
Solution:
  - Verify Gmail App Password (not main password)
  - Enable 2FA on Gmail: https://myaccount.google.com/apppasswords
  - Check EMAIL_HOST_USER and EMAIL_HOST_PASSWORD

Issue: "Database connection refused"
Solution:
  - Verify DATABASE_URL format
  - Check PostgreSQL credentials
  - Ensure database created
  - For Railway: railway services should show PostgreSQL

Issue: "500 Internal Server Error"
Solution:
  - Check ERROR logs: railway logs | grep ERROR
  - Check SECRET_KEY is set
  - Verify migrations: python manage.py migrate --check
  - Look for syntax errors: python manage.py check
